# Описание приложения для магазинов (App Store / RuStore / Google Play)

## Полное описание

В приложении собрано множество курсов для обучения собак.

Каждый курс — отдельная мини-история, которую вы проживаете вместе со своим хвостиком.

Уроки составлены и подробно расписаны профессиональными кинологами, чтобы было проще достичь целей.

Каждый урок разбит на дни, а дни — на маленькие шаги, по которым вы будете идти к компаньону своей мечты.

### Метод и подход

- Позитивное подкрепление — без давления и наказаний
- Подходит для щенков и взрослых собак

### Формат

- Короткие уроки по 5–15 минут в день
- Видео, иллюстрации и чек-листы
- Занимайтесь в своём темпе

### Темы курсов

- Послушание, команды, социализация
- Решение проблем: страхи, агрессия, тяга на поводке
- Трюки и совместные активности

### Прогресс и мотивация

- Отслеживание прогресса
- Достижения и награды
- Напоминания о занятиях

### Практичность

- Работа офлайн — занимайтесь без интернета
- Подходит для занятых людей
